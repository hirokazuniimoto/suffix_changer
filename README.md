# Features

フォルダの中にあるファイルの拡張子を一括で変えることができるライブラリです。

# Installation

Gitを使う場合は git clone https://github.com/hirokazuniimoto/YoURList-Et1-Public.git　
それ以外の場合はGitHubからZIPファイルとしてダウンロードしてください。

# Usage

Pythonで実行するプログラムです。

'''
cd suffix_changer
pyhton change_suffix.py [拡張子を変えたいフォルダのフルパス]
'''
